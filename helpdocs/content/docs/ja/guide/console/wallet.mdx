import { Callout } from 'fumadocs-ui/components/callout';

ここでは、内部残高管理、認証済み交換、およびコンプライアンスに準拠した招待記録を行うことができます。支払い方法は、管理者がシステム設定 -> 支払い設定で設定します。

<Callout type="warn">
  チャージ、引き換えコード、招待リンクは、合法的な承認済み利用シーンに適用されます。使用時には、承認範囲、アップストリームサービス規約、プラットフォーム規則、および現地の法令要件に準拠する必要があります。
</Callout>
